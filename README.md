This repository contains all of the key code for my STATS C116 final project (Spring 2026) that can be made publicly available. The project_workflow.R file contains the coding chunks used to run the models, evaluate their performance, and create visualizations, The beta_model_sparse_horseshoe.stan file was compiled for the first 358 gene run of the Beta hierarchical model with horseshoe priors on genes. The beta_inference_model_optimized.stan file was compiled for both following 15 and 50 gene runs with Gaussian priors on genes.
